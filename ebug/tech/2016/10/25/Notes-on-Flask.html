Notes on flask

Oct 25, 2016


Notes on using Flask, a microframework for creating apps in Python.

Intro to Flask

  • flask docs

Getting Started

create app environment

md flask_app
cd flask_app
virtual env

activate app from scripts

cd env
cd Scripts
activate

install flask in root

cd ..
cd ..
pip install flask

write app in python

from flask import Flask

app = Flask(__name__)

@app.route('/')
def index():
	return 'Hello World'

if __name__ == '__main__':
	app.run()

run app

python app.py

navigate to localhost:5000
